# .screenrc

## configuration for gnu screen

### disable startup message

    startup_message off

### use login shell

    defshell /usr/local/bin/bash

### disable wuff wuff bell

    bell_msg ""
    vbell off

### use 256 colors

    term xterm-256color

### increase scrollback

    defscrollback 2000

### start with UTF-8

    defutf8 on
